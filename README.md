> Aplicação web educacional para explorar o Problema do Clique, um problema clássico NP-Completo, com foco em teoria, implementação de algoritmos exatos e heurísticos, e simulações visuais da execução. O objetivo é comparar desempenho, qualidade de solução e discutir as limitações práticas de cada abordagem.

## 💻 Pré-requisitos

Antes de começar, verifique se você atendeu aos seguintes requisitos:

- Você instalou a versão mais recente de **Node.js 24+** e **npm 9+**.
- Você tem uma máquina **Windows, Linux ou macOS** (projeto compatível com os três sistemas).
- Você leu a documentação do **Vue 3**, **TypeScript** e **Vite**.

## 🚀 Instalando Click Problem

Para instalar o Click Problem, siga estas etapas:

Linux e macOS:

```bash
npm install
```

Windows (PowerShell):

```powershell
npm install
```

## ☕ Usando Click Problem

Para usar Click Problem, siga estas etapas:

```bash
npm run dev
```

Aplicação disponível em: http://localhost:5173

Comandos úteis:

```bash
npm run dev       # ambiente de desenvolvimento
npm run build     # build de produção
npm run preview   # pré-visualização da build
npm run lint      # validação estática (quando configurado)
```

## 📚 Escopo acadêmico

Este projeto aborda:

- Definição formal do Problema do Clique em grafos não direcionados.
- Versão de decisão: existe uma clique de tamanho k?
- Relação com classes de complexidade (NP, NP-Completo e NP-Difícil).
- Implementações práticas para:
- Algoritmo exato por força bruta;
- Algoritmos heurísticos para aproximação.
- Simulação visual da resolução e comparação de desempenho.
